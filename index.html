HTML Cheatsheet

  1. 1.HTML Boilerplate: Basic HTML structure 
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
</body>
</html>
  2. 2.Ordered List: Basic OL tag 
    <ol>
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
  3. 3.Unordered List: Basic UL tag 
    <ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
  4. 4.Link: Anchor tag for hyperlinks 
    <a href="https://example.com">Visit Example</a>
  5. 5.Image: Display an image 
    <img src="image.jpg" alt="Description">
  6. 6.Table: Basic table structure 
    <table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Cell 1</td>
    <td>Cell 2</td>
  </tr>
</table>
  7. 7.Input: Basic input field 
    <input type="text" placeholder="Enter text">
  8. 8.Button: Create a button 
    <button>Click Me</button>
  9. 9.Comment: Add a comment 
    <!-- This is a comment -->
  10. 10.Form: Basic form structure 
    <form action="#" method="post">
  <input type="text" name="name">
  <button type="submit">Submit</button>
</form>
  11. 11.Div: Container element 
    <div>Content here</div>
  12. 12.Span: Inline container 
    <span>Inline content</span>
  13. 13.Meta Tag: Set viewport 
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
  14. 14.Script Tag: Add JavaScript 
    <script>
  // JS code here
</script>
  15. 15.Style Tag: Add CSS 
    <style>
  /* CSS here */
</style>
