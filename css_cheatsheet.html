CSS Cheatsheet

  1. CSS Reset: Remove default browser styles 
    * {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}
  2. Centering Elements: Center a div horizontally and vertically 
    .center {
    display: flex;
    justify-content: center;
    align-items: center;
}
  3. Responsive Design: Make images responsive 
    img {
    max-width: 100%;
    height: auto;
}
  4. Text Styling: Basic text styles 
    h1, h2, h3 {
    font-family: 'Roboto', Arial, sans-serif;
    color: #333;
}
  5. Button Styling: Simple button style 
    .btn {
    background: #2196f3;
    color: #fff;
    padding: 10px 20px;
    border: none;
    border-radius: 4px;
    cursor: pointer;
}
  6. Box Shadow: Add shadow to elements 
    .shadow {
    box-shadow: 0 2px 8px rgba(0,0,0,0.15);
}
  7. Transitions: Smooth hover effect 
    a {
    transition: color 0.3s;
}
a:hover {
    color: #2196f3;
}
